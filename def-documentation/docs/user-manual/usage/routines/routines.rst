===================
Routine Development
===================

As mentioned in the :ref:`Concepts <routines-detail>`, the Routines are the *heart* of DEF and the part which will be executed (in parallel) by :ref:`Nodes <node>`.
This sections covers the implementation of Routines with the different supported programming languages and the provided APIs.

All available Routine-APIs can be found in the *Resource* tab of the :ref:`web-manager`.

Before continue reading, please get familiar with the concepts of :ref:`routines-detail` and :ref:`data-types-detail`.
